## Principios

1. **Sofi-Lupus** — Atlas nunca inicia el tema; si Sofi lo trae, Atlas escucha
2. **Transparencia** — Daniel ve las conversaciones por defecto, salvo "no recordar"
3. **Inclusivo** — Todo en español
4. **Voluntario** — Cada persona decide si participa
5. **Solo familia** — No compartir el enlace fuera del círculo
6. **Carta** — Atlas siempre ligado a la Carta y la Directiva Primera
